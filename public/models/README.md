# 3D Models Directory

Place your GLB/GLTF model files here.

## Required Models

| Filename | Description | Recommended Poly Count |
|----------|-------------|------------------------|
| `jeep.glb` | Old Mahindra Jeep (open-top, military green) | ~150k |
| `palm_tree.glb` | Coconut palm tree with fronds | ~20k |
| `tea_bush.glb` | Tea plantation bush cluster | ~5k |
| `kerala_house.glb` | Colonial wooden Kerala house | ~30k |

## Sources

- [Sketchfab](https://sketchfab.com) — Search "Mahindra Jeep", "Kerala House"
- [Poly.pizza](https://poly.pizza) — Free optimized low-poly
- [Quaternius.com](https://quaternius.com) — Free game-ready assets

## Tips

- Use `.glb` (binary) format for smaller file sizes
- Keep total scene < 50MB
- Include LODs if poly count is high
- Ensure proper UV unwrapping for textures

Without GLB files, the scene uses procedural primitives (still looks great!).
